## MODIFIED Requirements

### Requirement: 单节点聚焦优化面板

点击画布任一节点 SHALL 打开一个**锚定在该节点旁**的浮动面板;面板 SHALL 随画布平移 / 缩放保持锚定;面板 SHALL 在锚定节点被拖动或缩放时实时跟随其新位置 / 新尺寸;面板 SHALL NOT 为遮挡画布的居中模态。

#### Scenario: 点节点打开锚定面板
- **WHEN** 用户点击画布上的某个节点
- **THEN** 在该节点旁出现浮动优化面板,锚定到该节点

#### Scenario: 面板随画布跟随
- **WHEN** 画布发生平移或缩放
- **THEN** 面板位置跟随其锚定节点,保持相对关系

#### Scenario: 面板随节点位置 / 尺寸变化跟随
- **WHEN** 锚定的节点被用户拖动到新位置或缩放为新尺寸
- **THEN** 面板实时跟随,不掉队、不抖动

#### Scenario: 关闭与失效
- **WHEN** 面板被关闭、或其锚定节点被删除 / 移出视口
- **THEN** 面板消失,不残留悬空浮层
