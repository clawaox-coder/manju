## MODIFIED Requirements

### Requirement: 节点类型自适应的优化内容

面板内容 SHALL 依节点类型自适应:**内容节点**(剧本场 / 角色 / 分镜)呈现单元素优化,**枢纽节点**(ai-gen / video-out)呈现整体动作;5 类节点均可打开面板。分镜节点 SHALL 提供"重画这一镜"作为可执行入口(由 `canvas-image-generation` capability 提供后端);角色节点 SHALL 提供"AI 生成头像"作为可执行入口。

#### Scenario: 分镜节点单元素优化
- **WHEN** 打开分镜节点的面板
- **THEN** 提供"改对白 / 改时长 / 重画这一镜"及描述、参考图输入,且作用域**仅限该镜**

#### Scenario: 重画这一镜端到端可跑
- **WHEN** 用户在分镜面板触发"重画这一镜"
- **THEN** 系统生成图像并写回该 shot 的 `image_url`,画布刷新;不再返回未实现错误

#### Scenario: 剧本场节点单场优化
- **WHEN** 打开剧本场节点的面板
- **THEN** 提供针对该场的重写指令,执行后**仅改这一场**

#### Scenario: 角色节点单元素优化
- **WHEN** 打开角色节点的面板
- **THEN** 提供该角色名称 / 描述 / 设定的编辑与优化,以及"AI 生成头像"入口,作用域仅该角色

#### Scenario: 角色头像生成端到端可跑
- **WHEN** 用户在角色面板触发"AI 生成头像"
- **THEN** 系统生成图像并写回该 asset 的 `file_url`(覆盖旧值),画布角色节点刷新

#### Scenario: 枢纽节点整体动作
- **WHEN** 打开 ai-gen 或 video-out 节点的面板
- **THEN** 呈现其整体动作(生成全部分镜 / 渲染整片)而非单元素优化,且对该昂贵动作要求二次确认
